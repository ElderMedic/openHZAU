#############################################################################

The codes for the project are used in simulation experiments. The requirement 
of running environment is MATLAB R2013a or above versions.

The details of the files in the project directory are:
      main.m:The main and entry function
      AF_evaluate.m:The function of evaluating and judging the action of 
                       artificial fish
      simulatetest.m:The function of testing simulation
      prey.m:The function of prey
      aggregate.m:The function of aggregation
      follow.m:The function of following
